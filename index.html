História do Computador

O primeiro Computador

A invenção do computador não pode ser atribuída a apenas uma pessoa, a história do computador é longa, remontando às primeiras réguas de cálculo e às primeiras máquinas projetadas para facilitar a tarefa de aritmética para humanos. Os primeiros computadores surgiram como máquinas de cálculo lógico, devido às necessidades dos Aliados durante a Segunda Guerra Mundial. Para decodificar as transmissões dos lados em guerra, os cálculos tiveram que ser feitos rápida e constantemente, por esse motivo a Universidade de Harvard projetou o primeiro computador eletromecânico em 1944 baizado de MARK 1.

O que são Clients ?

Clients são os tipícos dispositivos conectados à internet dos usuários da web (por exemplo seu computador conectado ao seu WI-FI ou telefone conectado a rede móvel) e programas de acesso à web(geralmente um navegador como o Firefox ou Chrome), basicamente os clients na web são os dispositivos que estão consumindo informações na internet.

O que são Servers ?

Servers ou servidores são computadores que armazenam páginas, sites ou aplicativos. Quando o dispositivo de um cliente quer acessar uma página, uma cópia dela é baixada do servidor para a máquina do cliente para ser apresentada no navegador web do usuário.

O que são Linguagens de programação ?

Linguagem de programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software).

Para voltar ao: ÍNDICE

Linguagem Server-side

São linguagens executadas nos servidores, alguns exemplos são, ( PHP, RUBY, JAVA, JAVASCRIPT ). A programação do lado do servidor é muito útil porque nos permite fornecer com eficiência informações personalizadas para usuários individuais e, assim, criar uma experiência de usuário muito melhor.

Linguagem Client-side

São linguagens que são executadas no lado do cliente, ou seja, no computador do usúario onde só o navegador do mesmo vai entender e processar sem a necessidade de processamento do servidor.

O que é HTML ?

O HTML (HyperText Markup Language) não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura de uma pagina web e seu conteúdo. o HTML consiste de uma série de elementos que são usados para agrupar ou delimitar distintas partes do conteúdo para que ele pareça ou atue de determinada forma, por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista, imagens, tabelas e marcações.
